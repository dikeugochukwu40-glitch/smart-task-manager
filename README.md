# Smart Task Manager

A web app that helps users organize tasks with categories, deadlines, and filtering.

## Features
- Add and delete tasks
- Mark tasks as complete
- Filter by status
- Save data in browser storage

## Built With
- HTML
- CSS
- JavaScript
